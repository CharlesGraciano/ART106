Serial UI ( minus phase 2 )

  • This Serial UI phase is a continuation of our Micro phase but to another level. Now instead of only using arduino to communicate with the soldered hardware, now we are incorporating Processing to interact with our created device.

We ended the last phase by creating an enclosure for our breadboard. Within the box I also included Two neo pixel Led lights to my enclosure. These lights are unique because throughout the semester, I've learned that registration can help the voltage flow through to and from LED bulbs in order for them to function. These Neo pixels don't require resisters because they are incorporated within the device.
