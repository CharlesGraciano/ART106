MICRO

  • ADAFRUIT UNO MINI on breadboard.
  • Connected to a data cable, this adafruit version uno is designed to interact with mechanical devices and computer code.

ASSEMBLED BREADBOARD

  • The assembled BreadBoard has the following:
  • a speaker
  • two buttons
  •  pots
  • photocells
  • two leds
  • All of these materials are examples of both digital and analog input and output devices.
  • The code is being communicated through the arduino software to create UI.

ENCLOSED

  • I wanted to create a small box where I could keep everything in a tight space and it looked great! My only concern was when everything was together, I made it difficult to tweek or fix anything within the small space. It almost made it impossible to add anything else. I came to a conclusion that the next time i approach a scenario like this i will be using a bigger box.
